PicGo-CoreThe ultimate image uploading engine

Integrate image uploading into your build scripts, editors, or any Node.js workflow via CLI or API.

Guide
GitHub
PicGo Homepage

CLI automation

Upload images and get links with a single command. Fits perfectly into shell scripts and CI/CD pipelines.

Fast SDK integration

Don’t want to deal with authentication and upload logic? Bring in PicGo-Core and add image-host capability to your Node.js app in minutes.

Extensible ecosystem

Powered by a strong plugin system—add compression, watermarking, and other processing, or connect to any private image host.

bash
# Upload an image via CLI
npx picgo upload /path/to/image.png
js
// Seamlessly integrate into your Node.js workflow
const { PicGo } = require('picgo')
const picgo = new PicGo()

// Upload an image with a single line
picgo.upload(['/path/to/image.png'])
